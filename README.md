# Vlearn



## Installation

Clone repo:
```bash
git clone https://github.com/mrpottor/vlearn.git
```

Run the installer:
```bash
cd vlearn

Install using pip
```bash
pip3 install -r requirements.txt
```
or
```bash
pip3 install Jade-Application-Kit

sudo sh install.sh
```

## Usage

Open up menu, find "Vlearn" and click on it.
